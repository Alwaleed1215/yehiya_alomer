# مؤسسة يحي أحمد يحي ال عمر للتجارة - الموقع الرسمي

هذا المستودع يحتوي على الكود المصدري للموقع التعريفي الخاص بـ "مؤسسة يحي أحمد يحي ال عمر للتجارة"، وهي ورشة متخصصة في أعمال الديكور الخشبي والألمنيوم بحي اليرموك في مدينة الرياض.

## 🚀 مميزات الموقع (Features)
- **صفحة هبوط متكاملة (Single Page Application):** تصميم عصري وسريع الاستجابة (Responsive) يناسب جميع الشاشات.
- **متعدد اللغات (Multilingual):** يدعم 3 لغات (العربية، الإنجليزية، الأوردو) بنظام تبديل سلس.
- **الوضع الليلي والمضيء (Dark/Light Mode):** تبديل سلس بين الوضعين مع حفظ اختيار المستخدم.
- **تحسين محركات البحث المحلي (Local SEO):** - يحتوي على `Schema Markup` (JSON-LD) المخصص للأنشطة التجارية المحلية (LocalBusiness).
  - إحداثيات الموقع الجغرافي (Geo Meta Tags) لمدينة الرياض.
- **جاهز للتوثيق:** يتضمن كود إثبات الملكية لـ Google Search Console.
- **أزرار تواصل عائمة (Floating Action Buttons):** للاتصال المباشر والواتساب.
